# Directional Roadmap

This roadmap preserves future product directions only. It does not authorize implementation, change Joint's current MVP contract, or replace a purpose-specific implementation plan.

| Order | Feature | Description | Depends on | Status |
| --- | --- | --- | --- | --- |
| 1 | Budgets | Shared monthly household and expense-category spending targets compared with actual ledger spending. | Live categories, transactions, and dashboard reporting MVP. | Directional |
| 2 | Recurring Expenses | Reusable expense schedules whose occurrences require member review before creating ledger transactions. | Live expense categories, authenticated transaction actions, and transaction ledger. | Directional |
| 3 | Bills and Analysis | Manually maintained obligations, upcoming and overdue states, and explainable expected-versus-recorded analysis built after recurring-expense scheduling. | Recurring-expense scheduling; ledger, categories, and dashboard reporting. | Directional |
| 4 | Automatic Labelling | Optional consent-gated category suggestions from transaction text with no model-authorized financial mutation. | Live authenticated categories and manual transaction entry. | Directional |
| — | Transaction CSV Digest | Template-based read-only digest direction. | `docs/plans/transactions-statement-import.md` | Superseded by `docs/plans/transactions-statement-import.md`; no implementation order. |
